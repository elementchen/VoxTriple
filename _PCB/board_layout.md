# VoxTriple PCB 焊盘定位 & 布线指南

## 设计说明

这是一个简单的连线板（Breakout Board），所有元件都是 2.54mm 排针/排母或直插元件。  
在嘉立创EDA中，不需要导入网表，直接手动放置焊盘然后连线即可。

---

## 元件焊盘尺寸一览

| 元件 | 封装/规格 | 焊盘说明 |
|------|-----------|----------|
| MOD1 (ESP32 NodeMCU-32S) | 2×19P 排母，间距 2.54mm | 两排各19孔，排距 22.86mm (0.9") |
| MOD2 (INMP441) | 1×6P 排母，间距 2.54mm | 单排6孔 |
| SW1/SW2/SW3 (按键) | 6×6mm 轻触开关 | 2脚直插，间距约 4.5mm |
| LED1 (指示灯) | 3mm 红色 LED | 2脚直插，间距 2.54mm |
| R1 (限流电阻) | 220Ω，0805或1/4W直插 | 0805贴片间距 1.2mm，或直插间距 10mm |

## 模块一：ESP32 NodeMCU-32S (MOD1)

### 排母焊盘布局

```
        左排 (J1, 19脚)              右排 (J2, 19脚)
        间距 2.54mm                   间距 2.54mm
        ↑                            ↑
    ┌───● 1  3V3                GND  ●───┐
    │   ● 2  EN                 GPIO23●   │
    │   ● 3  GPIO36(VP)         GPIO22●   │
    │   ● 4  GPIO39(VN)        TXD/GP1●   │
    │   ● 5  GPIO34            RXD/GP3●   │
    │   ● 6  GPIO35             GPIO21●   │
    │   ● 7  GPIO32              GND ●   │
    │   ● 8  GPIO33(WS)         GPIO19●   │
    │   ● 9  GPIO25(BCK)   GPIO18(BTN1)●  │
    │   ●10  GPIO26(LED)         GPIO5●   │
    │   ●11  GPIO27        GPIO17(DATA)●  │
    │   ●12  GPIO14(BTN2)  GPIO16(BTN3)●  │
    │   ●13  GPIO12  ⚠            GPIO4●  │
    │   ●14  GND             GPIO0(BOOT)● │
    │   ●15  GPIO13               GPIO2●  │
    │   ●16  D2(GPIO9)           GPIO15●  │
    │   ●17  D3(GPIO10)        D1(GPIO8)● │
    │   ●18  CMD(GPIO11)       D0(GPIO7)● │
    └───●19  5V(VIN)              GND ●───┘
        2.54mm × 18 = 45.72mm (排高)
```

**关键尺寸：**
- 焊盘孔径：1.0mm
- 焊盘外径：1.8mm
- 排针间距：2.54mm（列内）
- 两排间距：22.86mm（约 0.9 英寸）
- 单排总长：45.72mm（18 × 2.54mm）

### 在嘉立创EDA中放置
```
1. 放置焊盘 → 选择"通孔焊盘"，孔径1.0mm，外径1.8mm
2. 放置19个焊盘，间距2.54mm，垂直排列（Y方向）
3. 复制该排到右侧22.86mm处
4. 在左侧排上方标注"J1"，右侧排上方标注"J2"
```

## 模块二：INMP441 MEMS 麦克风 (MOD2)

### 排母焊盘布局

```
    ● 1  SCK  ←──→ GPIO25 (I2S_BCK)
    ● 2  WS   ←──→ GPIO33 (I2S_WS)
    ● 3  L/R  ←──→ GND
    ● 4  SD   ←──→ GPIO17 (I2S_DATA)
    ● 5  GND  ←──→ GND
    ● 6  VDD  ←──→ 3.3V
    间距 2.54mm
    排高 12.7mm（5 × 2.54mm）
```

**关键尺寸：**
- 焊盘孔径：1.0mm
- 焊盘外径：1.8mm
- 排数：单排6脚
- 间距：2.54mm
- 排总长：12.7mm

### 在嘉立创EDA中放置
```
1. 放置焊盘 → 通孔焊盘，孔径1.0mm，外径1.8mm
2. 放置6个焊盘，间距2.54mm，垂直排列
3. 在排上方标注"INMP441"
```

## 模块三：按键 (SW1, SW2, SW3)

### 轻触开关焊盘

```
    ●──────●      间距 ~4.5mm
    1      2      (6×6mm 轻触开关的标准脚距)
```

**关键尺寸：**
- 焊盘孔径：1.2mm
- 焊盘外径：2.0mm
- 两脚间距：~4.5mm（6×6mm轻触开关标准）
- 开关本体：6mm × 6mm

### 在嘉立创EDA中放置
```
1. 放置两个通孔焊盘，间距4.5mm
2. 对于每个按键重复放置
3. 标注为 SW1、SW2、SW3
```

## 模块四：LED 指示灯 (LED1)

```
    ●───R1───●──────●
  GPIO26   220Ω    LED 阳极(A)→阴极(K)→GND
```

**关键尺寸：**
- LED 焊盘孔径：0.8mm
- LED 焊盘外径：1.5mm
- LED 脚间距：2.54mm
- 电阻 R1：可用 0805 贴片（间距1.2mm）或 1/4W 直插（间距10mm）

## 布线连接表

| 网路名 | 起点 (MOD1) | 终点 |
|--------|------------|------|
| **3V3** | MOD1 左排 Pin 1 (3V3) | MOD2 Pin 6 (VDD) |
| **GND** | MOD1 左排 Pin 14, 右排 Pin 1/7/19 (GND) | MOD2 Pin 2 (GND), MOD2 Pin 3 (L/R), SW1 Pin 2, SW2 Pin 2, SW3 Pin 2, LED 阴极(K) |
| **I2S_BCK** | MOD1 左排 Pin 9 (GPIO25) | MOD2 Pin 1 (SCK) |
| **I2S_WS** | MOD1 左排 Pin 8 (GPIO33) | MOD2 Pin 2 (WS) |
| **I2S_DATA** | MOD1 右排 Pin 11 (GPIO17) | MOD2 Pin 4 (SD) |
| **BTN1** | MOD1 右排 Pin 9 (GPIO18) | SW1 Pin 1 |
| **BTN2** | MOD1 左排 Pin 12 (GPIO14) | SW2 Pin 1 |
| **BTN3** | MOD1 右排 Pin 12 (GPIO16) | SW3 Pin 1 |
| **LED_NET** | MOD1 左排 Pin 10 (GPIO26) | R1 一端 |
| **LED_ANODE** | R1 另一端 | LED 阳极(A) |

## 在嘉立创EDA中的操作步骤

1. **新建PCB**：文件 → 新建 → PCB
2. **放置ESP32焊盘**：
   - 用"焊盘"工具放置 1.0mm 通孔焊盘
   - 阵列复制：19行 × 2.54mm间距 → 得到左排
   - 右侧 22.86mm 处复制 19 个 → 得到右排
3. **放置INMP441焊盘**：
   - 6个通孔焊盘，2.54mm间距，单排
4. **放置按键/ LED/电阻焊盘**：
   - 按键：2个焊盘，间距 4.5mm
   - LED：2个焊盘，间距 2.54mm
   - 电阻：2个焊盘（0805或直插）
5. **布线**：按上表连接对应的焊盘
6. **铺铜**：底层大面积 GND 铺铜
7. **导出 Gerber**：发给嘉立创打样
