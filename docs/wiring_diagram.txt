================================================================================
                              I2S配置参数
================================================================================

采样率:     16000 Hz (16kHz)
位宽:       16-bit
声道:       单声道 (Mono)
格式:       标准I2S (MSB对齐)
时钟主频:   16kHz * 16bit * 2 = 512kHz (BCK)

================================================================================
                              按钮检测参数
================================================================================

消抖时间:   50ms
检测方式:   下降沿中断 + 软件消抖
上拉电阻:   内部上拉 (10kΩ)
触发电平:   低电平有效 (按下时接地)

================================================================================
                              蓝牙参数
================================================================================

设备名称:   ESP32_BT_MIC
蓝牙模式:   BTDM (双模)
经典蓝牙:   HFP AG (免提音频网关)
BLE:        GATT Server
配对码:     1234
广播间隔:   96ms

================================================================================
                              电源要求
================================================================================

ESP32开发板: USB 5V供电 (通过CP2102)
INMP441模块: 3.3V供电 (从ESP32的3.3V引脚)

注意: INMP441是3.3V器件，不要直接连接到5V！

================================================================================
                              注意事项
================================================================================

1. 确保所有GND引脚共地
2. INMP441的L/R引脚接地选择左声道
3. 按钮使用内部上拉，无需外部电阻
4. GPIO 32是仅输入引脚，适合I2S数据输入
5. 避免使用GPIO 6-11 (连接到SPI Flash)
6. 避免使用GPIO 34-39 (仅输入，无上拉)
